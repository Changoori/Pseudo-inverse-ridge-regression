# Pseudo-inverse Ridge Regression and Reconstruction Verification

This repository contains the MATLAB and Python code used for the verification analyses presented in the comment on Extended-DeepGS.

- The MATLAB script reproduces the pseudo-inverse ridge-regression analyses reported in Tables 2 and 3.
- The Python script reproduces the loam-case soil-moisture reconstruction and error comparison presented in Figure 1.

## MATLAB analysis

The MATLAB script performs the pseudo-inverse ridge-regression sensitivity analysis for the sandy-loam, loam, and silt-loam cases.

### Requirements

- MATLAB
- No additional toolbox is required unless otherwise specified in the script.

### How to run

1. Download or clone this repository.
2. Open MATLAB.
3. Set the repository root, `Pseudo-inverse-ridge-regression/`, as the current working directory.
4. Open the following script:

```text
matlab/pseudo_inverse_ridge_regression.m
```

5. In the data-loading section, activate the `load` command corresponding to the soil case to be analyzed.
6. Run the script.

Example:

```matlab
% Loam
% load(fullfile('data', 'loam', 'th.txt'))

% Sandy loam
% load(fullfile('data', 'sandy_loam', 'th.txt'))

% Silt loam
load(fullfile('data', 'silt_loam', 'th.txt'))
```

The script reproduces the active-term selections and coefficient estimates reported in Tables 2 and 3 of the associated manuscript.

## Python reconstruction analysis

The Python script performs the loam-case soil-moisture reconstruction and calculates the spatiotemporal reconstruction error used in Figure 1.

### Requirements

The required Python packages and environment information are provided in:

```text
environment.txt
```

### How to run

1. Download or clone this repository.
2. Install the Python environment specified in `environment.yml`.
3. Open a terminal in the repository root.
4. Move to the Python directory:

```bash
cd python
```

5. Run the reconstruction script:

```bash
python "Reconstruction figure.py"
```

The Python script uses the following input file:

```text
python/th.txt
```

This file contains the loam-case soil-moisture data used to generate the reconstructed field, reference field, and reconstruction-error distribution presented in Figure 1.

## Data provenance

The soil-moisture data used in this verification analysis were obtained from the Zenodo repository accompanying Song et al. (2025):

> Song, W. (2024). *extended-deepgs* [Data set]. Zenodo.  
> https://doi.org/10.5281/zenodo.14551650

The sandy-loam, loam, and silt-loam datasets were used in the MATLAB regularization-sensitivity analysis.

The loam `th.txt` dataset was also used in the Python reconstruction and error-comparison analysis presented in Figure 1.

The original soil-moisture data were not generated by the authors of the present comment. The files are included in this repository to support reproducibility and remain subject to the license and reuse conditions of the original Zenodo record.

## Reproducibility

The MATLAB and Python analyses were performed using the data and scripts included in this repository.

The MATLAB analysis reproduces:

- the regularization-sensitivity analysis for three heterogeneous soil cases;
- the active-term selections reported in Table 2; and
- the governing-equation coefficients reported in Table 3.

The Python analysis reproduces:

- the loam-case reconstructed soil-moisture field;
- the corresponding reference field; and
- the spatiotemporal reconstruction-error distribution presented in Figure 1.

For reproducibility testing, users are encouraged to download a clean copy of the repository and run the scripts according to the instructions above.

## Computational environment

The analyses reported in the manuscript were performed on a workstation equipped with:
